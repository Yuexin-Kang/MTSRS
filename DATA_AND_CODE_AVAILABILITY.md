# Data and Code Availability

Author: Yuexin Kang

The computational data used in this study are synthetically generated instances. This repository provides instance scale tables, generated instance data, order-tote requirement matrices, main system parameters, tabulated experimental results, cleaned computational records, and detailed pseudocode for the instance-generation and experimental workflows.

The generated instances and computational records are organized by experiment type, including main makespan experiments, total-completion-time experiments, sensitivity analyses, and emergency-reservation experiments.

The full implementation code is not publicly released at this stage because the computational results depend on solver versions, software dependencies, and server configurations, and parts of the implementation are being extended in ongoing research. Instead, this repository provides detailed algorithmic descriptions, parameter settings, generated instance data, experimental tables, and supporting computational records to allow readers to verify the experimental design and reported results.

# Experimental Tables

Author: Yuexin Kang

This folder contains tabulated experimental results from the numerical study. The files are organized by experiment type.

| File | Content |
|---|---|
| `main_makespan_results.xlsx` | Main comparison results under the makespan objective. |
| `main_tct_results.xlsx` | Main comparison results under the total-completion-time objective. |
| `sensitivity_results.xlsx` | Sensitivity analysis results under makespan and total-completion-time objectives. |
| `emergency_reservation_results.xlsx` | Emergency-reservation experiment results. |

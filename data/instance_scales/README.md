# Instance Scales and Experiment Grids

Author: Yuexin Kang

This folder contains machine-readable tables for the instance scales and experiment grids used in the numerical study.

## Files

| File | Description |
|---|---|
| `main_instances.csv` / `main_instances.xlsx` | Main small-, medium-, and large-scale instance groups used in the primary comparison experiments. |
| `drl_alns_comparison_instances.csv` / `drl_alns_comparison_instances.xlsx` | Instance groups used to compare DRL-HALNS with traditional ALNS. |
| `sensitivity_grids.csv` / `sensitivity_grids.xlsx` | Parameter grids used in the sensitivity analyses for layout, workload, robots, and stations. |
| `emergency_experiment_grid.csv` / `emergency_experiment_grid.xlsx` | Experimental-factor grid used in the emergency-reservation experiments. |

The CSV files are provided for machine-readable use. The XLSX files provide the same information in a spreadsheet format.

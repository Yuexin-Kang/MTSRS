# Instance Scales and Experiment Grids

Author: Yuexin Kang

This folder contains CSV files that describe the instance scales and experiment grids used in the numerical study.

## Files

| File | Description |
|---|---|
| `main_instances.csv` | Main small-, medium-, and large-scale instance groups used in the primary comparison experiments. |
| `drl_alns_comparison_instances.csv` | Instance groups used to compare DRL-HALNS with traditional ALNS. |
| `sensitivity_grids.csv` | Parameter grids used in the sensitivity analyses for layout, workload, robots, and picking stations. |
| `emergency_experiment_grid.csv` | Experimental-factor grid used in the emergency-reservation experiments. |

All files are provided in CSV format for readability and machine-readable use.
